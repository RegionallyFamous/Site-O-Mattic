## Reusable Build Pattern

For all of these, keep the Blueprint self-contained and use core Cover, Image, Gallery, Columns, Buttons, Group, Details, Navigation, List, Paragraph, and Heading blocks. CTAs should be real `tel:` or `mailto:` links, in-page anchors should target real sections, and the visual system should come from theme/global styles rather than plugin CSS.
