## Prompt

Use case: production favicon for an internal sales gallery catalog page

Create one square favicon-style bitmap icon for Site-O-Mattic, a riso-bright catalog of polished WordPress Playground blueprint demo sites. No words, no letters, no initials, no readable text, no fake UI text, no WordPress logo, no watermarks.

Subject: one single bold cobalt-blue rounded browser tile or demo-card silhouette, front-facing with a slight playful tilt, with one oversized sunny-yellow play triangle cut into or sitting on the tile. Add one tiny tomato-red pin dot and one small mint spark only if they remain bold. The icon must read as one simple launch/demo mark, not a detailed web page.

Composition: centered square mark with generous padding on a clean white/off-white background, strong silhouette readable at 64px, 32px, and 16px, clean edges. Use big simple shapes. Avoid multiple tiny cards, detailed browser chrome, tiny text, miniature screenshots, thin strokes, or gradients that blur the shape.

Style: high-end riso/screenprint bitmap illustration, bold flat shapes, subtle halftone grain, crisp sticker-like icon.

Palette: cobalt blue, sunny yellow, deep navy/black ink, with only small tomato red and mint accents. Avoid brown, tan, beige, sepia, gray mud, and purple dominance.

Visual acceptance: v2 accepted after 64px, 32px, and 16px preview because it reads as one blue launch tile with a yellow play cue. The previous multi-detail browser-card favicon was rejected as too busy at 16px.
