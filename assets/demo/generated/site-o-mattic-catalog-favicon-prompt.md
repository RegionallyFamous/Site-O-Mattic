## Prompt

Use case: production favicon for an internal sales gallery catalog page

Create one square favicon-style bitmap icon for Site-O-Mattic, a riso-bright catalog of polished WordPress Playground blueprint demo sites. No words, no letters, no initials, no readable text, no fake UI text, no WordPress logo, no watermarks.

Subject: one bold tilted website/demo card or browser window with a large checkmark/play-arrow cue, plus one small riso spark or pin accent. It should instantly feel like a catalog of demo websites, not a generic app icon.

Composition: centered square mark with generous padding, white or very light background, strong silhouette readable at 64px, 32px, and 16px, clean edges. Avoid multiple tiny cards, detailed browser chrome, or tiny text.

Style: high-end riso/screenprint bitmap illustration, bold flat shapes, subtle halftone grain, crisp sticker-like icon.

Palette: cobalt blue, teal, tomato red, sunny yellow, deep navy/black ink, clean white/off-white paper. Avoid brown, tan, beige, sepia, gray mud, and purple dominance.

Visual acceptance: accepted after 64px, 32px, and 16px preview; icon reads as a single demo-card/browser tile plus launch cue, with stronger browser-tab silhouette than the earlier multi-card version.
